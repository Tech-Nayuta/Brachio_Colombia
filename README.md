# Brachio_Colombia
ブラキオカップ 2021.3.15 ~ 2021.3.17

グループごとで1人1人のプロフィールを登録し、思い出として残せる名鑑アプリを開発しました。

言語はSwift, データベースにCloudFirestore, ストレージにFirebaseStorageを使用しました。
